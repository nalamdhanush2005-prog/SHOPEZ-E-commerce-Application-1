🛒 SHOPEZ – E-Commerce Web Application

📌 Project Overview

SHOPEZ is a full-stack e-commerce web application that allows users to browse products, add items to cart, and securely place orders. It demonstrates core software engineering concepts like authentication, database management, REST APIs, and responsive UI design.

🚀 Features

👤 User Registration & Login (Authentication)

🛍️ Product Listing & Search

🛒 Add to Cart / Remove from Cart

💳 Checkout System (basic or simulated payment)

📦 Order Management

🛠️ Admin Panel (Add/Edit/Delete Products)

📱 Responsive UI (Mobile-friendly)

🧑‍💻 Tech Stack

Choose one stack (recommended below):

Option 1 (Popular Full Stack)

Frontend: HTML, CSS, JavaScript / React

Backend: Node.js, Express.js

Database: MongoDB

Option 2 (Simple for Mini Project)

Frontend: HTML, CSS, Bootstrap

Backend: Python (Flask) / PHP

Database: MySQL
